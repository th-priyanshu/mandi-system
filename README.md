# 🌾 Mandi Ledger System

A Full-Stack MERN application to manage Mandi trading operations, track live prices, and handle partial/full payments.

## 🚀 Features
- Live Mainpuri Mandi rate tracking (Govt. API)
- Purchase (Inward) & Sales (Outward) automated inventory loops
- Partial payment management for pending dues
- 24/7 Live Deployment on Vercel and Render

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Cloud)
